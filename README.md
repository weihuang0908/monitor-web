## Install in windows

使用带有管理员权限的命令行
安装过程比较漫长，耐心等待

 1/ 下载安装node
 2/ node install -g bower
 3/ node install -g grunt-cli
 4/ 进入程序目录,如
    cd park-prog/
 5/ node install grunt
 6/ node install
 7/ bower install
 
 安装结束之后
   grunt serve
 理论上会自动打开浏览器0.0.0.0:9000,如果挂掉了就在浏览器地址栏输入localhost:9000
 附：切换磁盘,如从c:/切换到e:/,只需输入e:
